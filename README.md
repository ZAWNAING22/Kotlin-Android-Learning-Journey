## **Phase 1 вҖ“ Kotlin Basics** рҹҹў

* **Variables and Data Types** вҖ“ `val` vs `var`, `Int`, `Double`, `String`, `Boolean`
* **Operators** вҖ“ Arithmetic, Comparison, Logical, Assignment
* **Functions** вҖ“ Normal functions, return types, function overloading
* **Lambda Functions** вҖ“ Anonymous functions for concise code
* **Higher-Order Functions** вҖ“ Functions as parameters or return values
* **Extension Functions** вҖ“ Adding functions to existing classes
* **Loops and Conditionals** вҖ“ `for`, `while`, `do-while`, `if`, `when`
* **Null Safety** вҖ“ Nullable types (`?`), safe calls (`?.`), Elvis operator (`?:`), non-null assertion (`!!`)
* **Collections** вҖ“ Arrays, Lists, Sets, Maps (mutable and immutable)
* **Console Input/Output** вҖ“ `print`, `println`, string templates (`$` and `${}`)
* **Exception Handling** вҖ“ `try`, `catch`, `finally`, throwing exceptions

---

## **Phase 2 вҖ“ Object-Oriented Programming** рҹҸӣпёҸ

* **Classes** вҖ“ Defining classes, objects
* **Constructors** вҖ“ Primary and secondary constructors
* **Inheritance** вҖ“ Extending classes and overriding methods
* **Abstract Classes** вҖ“ Classes with abstract methods
* **Interfaces** вҖ“ Defining contracts for classes
* **Data Classes** вҖ“ Classes for holding data with auto-generated functions
* **Sealed Classes and Sealed Interfaces** вҖ“ Restricting class hierarchies
* **Exception Handling in Classes** вҖ“ Handling errors inside methods (optional but useful)
